JS로 요소를 만드시오.
        1. 너비, 높이를 입력받은 값으로 지정
        2. 해당 요소 안에 작성되는 내용도 입력받은 값으로 지정
        3. 글씨색도 입력받은 값으로 지정
        4. 요소 안의 글씨도 입력받은 값의 위치에 오도록 지정
        => 입력받은 값 == input을 의미
        5. 적용 버튼 클릭 시 위 내용대로 요소가 만들어지도록! 연습!
        
        너비 : input
        높이 : input
        출력할 내용 : input => ex) Hello
        => 이걸로 요소가 나와야 함

        예를 들면 width = 200, height = 300 그 안에 요소가 들어가야 함
